What Was Fixed:

The original wave positioning issues were solved by:

  • Creating a dedicated container for the waves with absolute positioning
  • Setting proper height for the wave container (150px)
  • Adjusting the background-size to match the container height
  • Ensuring the wave container is positioned at the bottom of its parent
  • Adding overflow: hidden to prevent horizontal scroll
/* Key CSS Fixes */
.waves-container {
  position: absolute;
  bottom: 0;
  left: 0;
  width: 100%;
  height: 150px; /* Fixed height */
  overflow: hidden;
}

.wave1 {
  background-size: 50% 150px; /* Matches container height */
}

.wave2 {
  background-size: 50% 130px;
}

.wave3 {
  background-size: 50% 100px;
}
